模版式表单------- forms1 works!

  • 对象转json格式显示:{{myForm.value | json}}
  • 访问 passwordGROUP >> {{myForm.value.passwordGROUP | json}}
  • 获取模版变量 myEMAIL >> {{ myEMAIL.value}}
    • user Name is required.
    Name must be at least 3 characters long.
    invalid input
  • 引入了FormsModule模块后，angular自动在form标签里放入ngForm标签,并接管from表单， submit不起作用。。。 ngForm也可以放div里，就不需要form 标签
  • 如果ngNoForm 则表单不被angular接管
  • 每个接收数据的表单，必须有 ngModel name=" xxx " ， xxx作为 myForm 对象的 一个属性
  • 模版变量的 绑定 需要在表单里声明 #myEMAIL ="ngModel",访问则{{ myEMAIL.value}}
  • ngModelGroup 用于组织表单属性, 仅需要在几个表单外层标签上声明 ngModelGroup = "passwordGROUP"， json 数据遍自动加属性分组
